Text Input

Single-line text fields — the building block of any form.

Syntax
<!-- Basic text field -->
<input type="text"
       name="username"
       size="35"
       maxlength="50"
       value="default text" />

<!-- Email field (validates format) -->
<input type="email"
       name="email"
       placeholder="you@example.com" />

<!-- Password (masks input) -->
<input type="password"
       name="pass"
       minlength="8" />
AttributePurpose
sizeVisible width in characters
maxlengthMax characters allowed
valuePre-filled default text
placeholderHint text (disappears on focus)
requiredMust be filled before submit

Textarea

Multi-line text input — sized by columns and rows.

Syntax
<!-- cols = width, rows = height -->
<textarea
  name="message"
  cols="50"
  rows="5"
  placeholder="Write here..."
>Default text here</textarea>
AttributePurpose
colsWidth in characters (~pixels)
rowsVisible lines (height)
wrapsoft / hard — how text wraps
resizeCSS: resize: none|both|vertical

Radio Buttons

Mutually exclusive choices — same name = one group.

Syntax
<!-- All share name="plan" → only one can be selected -->
<input type="radio" name="plan"
       value="free" checked /> Free

<input type="radio" name="plan"
       value="pro" /> Pro

<input type="radio" name="plan"
       value="enterprise" /> Enterprise
⚠ Same NAME = one group
⚠ Only ONE selectable
⚠ checked sets the default

Checkboxes

Multiple simultaneous selections — each is independently toggled.

Syntax
<!-- Multiple can be checked at once -->
<input type="checkbox"
       name="skills"
       value="html"
       checked /> HTML

<input type="checkbox"
       name="skills"
       value="css" /> CSS

<input type="checkbox"
       name="skills"
       value="js" /> JavaScript
✓ Multiple can be selected
✓ Each has its own value
✓ checked sets default on

Select / Dropdown

Compact choice lists — use size to toggle between dropdown and scrolling list.

Syntax
<!-- size="1" → dropdown (default) -->
<select name="country" size="1">
  <option value="ng">Nigeria</option>
  <option value="uk" selected>UK</option>
</select>

<!-- size="4" → scrolling list -->
<select name="skills"
        size="4"
        multiple>
  <option>React</option>
  <option>Vue</option>
  <option>Angular</option>
</select>
AttributeEffect
size="1"Collapses into a dropdown
size="n"Shows n options as scrolling list
multipleAllows multi-select (Ctrl+click)
selectedPre-selects an option

Table Basics

The anatomy of an HTML table — rows, headers, data cells, and semantic grouping.

Core Structure
<table border="1">

  <!-- Optional caption -->
  <caption>Student Results</caption>

  <!-- Header rows -->
  <thead>
    <tr>
      <th>Name</th>
      <th>Score</th>
    </tr>
  </thead>

  <!-- Body rows -->
  <tbody>
    <tr>
      <td>Alice</td>
      <td>88</td>
    </tr>
  </tbody>

  <!-- Footer row -->
  <tfoot>
    <tr>
      <td>Average</td>
      <td>79</td>
    </tr>
  </tfoot>

</table>
Tag Reference
TagRole
<table>Outer container for the entire table
<caption>Visible title above the table
<thead>Groups header rows (semantic)
<tbody>Groups the main data rows
<tfoot>Groups footer rows (totals/averages)
<tr>Table Row — holds cells
<th>Header Cell — bold + centred by default
<td>Data Cell — regular content

Table Attributes

Controlling spacing, borders, alignment, and width.

Attribute Syntax
<!-- Table-level attributes -->
<table
  border="1"        ← border thickness
  cellpadding="8"  ← space inside cells
  cellspacing="2"  ← space between cells
  width="80%"      ← table width
  align="center"   ← table alignment
>

<!-- Cell-level attributes -->
<td
  align="right"    ← horizontal align
  valign="top"    ← vertical align
  width="150"     ← column width
  bgcolor="#eee"  ← background colour
>

align <table>, <td>, <th> left|center|right
valign <td>, <th> top|middle|bottom
bgcolor <table>, <tr>, <td> #hex or name

Spanning Cells

colspan merges across columns, rowspan merges down rows.

Syntax
<!-- colspan: cell takes 3 columns -->
<td colspan="3">Spans 3 columns</td>

<!-- rowspan: cell takes 2 rows -->
<td rowspan="2">Spans 2 rows</td>

<!-- Combined -->
<td colspan="2" rowspan="2">
  Big merged cell
</td>

<!-- RULE: when a cell has colspan="3",
  the next 2 cells in that row are
  OMITTED in the HTML -->
⚠ colspan = skip N-1 cells in that row
⚠ rowspan = skip that cell in next N-1 rows
